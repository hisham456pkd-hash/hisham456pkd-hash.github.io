What Is a QR Code and How Does It Work?

A QR code (Quick Response code) is a type of barcode that stores information such as links, text, or contact details.

How QR codes work

When scanned using a phone camera, the code is decoded instantly and shows the stored data.

Common uses

  • Website links
  • Wi-Fi sharing
  • Payments and menus
